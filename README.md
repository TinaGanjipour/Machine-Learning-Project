# Predicting Heart Disease Using Machine Learning

This project was my bachelor thesis and was one of my first introductions to both machine learning and bioinformatics. Funny enough, it was also one of the projects that helped me realize how much I enjoy bioinformatics and data-driven problem solving.

In a nutshell, this project explores whether machine learning models can predict the presence of heart disease based on clinical features such as age, sex, chest pain type, blood pressure, cholesterol, and other medical attributes. The notebook walks through the full workflow: problem definition, data exploration, feature analysis, model training, evaluation, and comparison of different classifiers including Logistic Regression, K-Nearest Neighbors, and Random Forest.
